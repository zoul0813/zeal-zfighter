# Z-Fighter for Zeal 8-bit Computer

[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Z-Fighter is a side-scrolling space shooter for Zeal 8-bit Computer.

Z-Fighter contains background music (provided by [Zeal Music Ttrack (ZMT)](https://github.com/zoul0813/zeal-music-tracker), sound effects, and even the occassional asteroid/meteor you need to avoid.

## Controls

Z Fighter supports both Keyboard and SNES controller input.

| Action | SNES   | Keyboard |
| ------ | ------ | -------- |
| Up     | Up     | Up       |
| Down   | Down   | Down     |
| Left   | Left   | Left     |
| Right  | Right  | Right    |
| Start  | B      | Z/Space  |
|        | A      | X        |
|        | Y      | A        |
|        | X      | S        |
| Quit   | Select | R Shift  |
|        | Start  | Enter    |
|        | L      | Q        |
|        | R      | W        |
